# AWS EC2 Instance Launch Lab Report

## Objective
The purpose of this lab was to successfully launch and configure an Amazon Elastic Compute Cloud (EC2) instance using the AWS Management Console.

---

# Step 1: Launching an EC2 Instance

The EC2 launch wizard was used to configure a new virtual server instance. During the setup process, the following configurations were completed:

- Assigned the instance name as **Web Server**
- Selected the **Amazon Linux 2023 AMI**
- Chose the **t3.micro** instance type
- Configured the key pair for secure login access
- Configured networking and security group settings
- Enabled storage configuration
- Added user data for web server setup

The screenshot below shows the EC2 instance launch configuration page before deployment.

# Step 2: Successfully Launching the Instance

After reviewing all configurations, the **Launch Instance** button was selected. AWS successfully initiated the EC2 instance deployment.

The confirmation screen below verifies that the EC2 instance was successfully launched.

![EC2 Launch Success](screencapture-us-west-2-console-aws-amazon-ec2-home-2026-05-24-20_14_22.png)
